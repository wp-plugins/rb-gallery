A one-page carousel photo gallery plugin for your WordPress site.

== Description ==

If you are looking for an simple, easy to use photo gallery plugin, then RB Gallery is for you.

Setting up your gallery literally takes a couple of minutes using the simple (one-page) user interface and settings panel.

Below is a list of the features offered in this photo gallery plugin:

= Easy gallery creation =
* Upload your photos to the gallery in GIF, JPG, or PNG format.
* Add as many images as you want to your photo gallery
* Edit the images and other image properties (example: name, description etc).
* Exclude/include photos which are part of a album/gallery.
* Customize stylish front end albums.
* Ability to sort your gallery images.
* Flexible gallery shortcodes which you can apply to any post or page to insert your gallery.

Visit the [RB Gallery Plugin](http://www.ronniebailey.net) for more information.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the 'wprb-gallery.zip' file from the Plugins -> Add New page in the WordPress administration panel.
2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

Go to the RB Gallery menu under 'Settings' and start creating your photo albums. 

== Frequently Asked Questions ==

= Can I upload jpg or png image files? =

Yes, you can upload GIF, PNG, and JPG files

= Can I upload lots of photos to my albums? =

Yes, but the more photos you upload, the more resources your web server will be using so you may want to seperate them out into more albums to get better performance.
